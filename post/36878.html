AI作画

说在前面

这是在2021上半年前，GAN自动生成动漫人物也是我们社科研讨课的一个调研方向。由于2022上半年 AI作画疯狂内卷，涌现了很多算法。

我们2021年的评估报告：“由于现在的自动生成动漫人物的相关研究刚刚起步，尚未推广到产业领域，而且生成质量较低，尚且不能达到受众的要求……应用主要在仅生成动漫图像的较小众的娱乐方面。……现有的研究基本上是围绕动漫人物头部的自动生成展开的。”

当时的预期：“目前，我们无法真正控制我们想要生成的特征，例如头发颜色、眼睛颜色、人物风格、发型和配饰，如果能和针对其他部分，如身体形态、周边环境、服装等建立的GAN生成器进行结合，可以更好地提高动漫和漫画作品的效率和实用性。”

另外在绘画辅助方面：有自动上色，自动提取人物线稿，更改已有图像风格等，以及live2d的自动生成！

算法

GAN架构原理：两个网络的相互博弈

  • 生成器：利用随机噪声产生假的动漫人物图片，并欺骗鉴别器为真实样本
  • 鉴别器：尝试从真样本和假样本中检测生成的样本

改进的GAN–StyleGAN：允许控制生成的样本的不同层次的细节水平，制住输出的图片的具体特征

  • 将生成器划分为不同的层次，它们能够控制图像的不同视觉特征
  • 生成器中添加了映射网络，将输入向量编码为中间向量，传给生成网络得到18个控制向量，使得该控制向量的不同元素能够控制不同的视觉特征。
  • 映射网络可以使得18个控制向量存在特征纠缠（某些控制内容的相互影响）
  • 不同层次有单独的输入向量，控制不同的视觉特征（粗糙或细节）
  • 阶段技巧训练代表性不足数据的处理，添加随机变化的相关操作

参考：NVIDIA发布StyleGAN论文“A Style-Based Architecture for GANs”。

Colab 是谷歌内部类 Jupyter Notebook 的交互式 Python 环境，免安装快速切换 Python 2和 Python 3 的环境，支持 Google 全家桶（TensorFlow、BigQuery、GoogleDrive 等），支持 pip

注册账号：https://huggingface.co/join ，提前注册谷歌账号和colab，运行时类型为GPU加速

CLIP

文字生成图像的模型，基于对比的图片-文本学习的跨模态预训练模型，由OpenAI于去年1月发布。

  • 输入文本关键词
  • 基于训练数据集，从一堆噪点中把这个图像反推出来
  • 不断进行文本与图像的匹配
  • 逐步消除噪声，添加细节

Waifulabs

古早的生成二次元老婆头像的应用，2021年我们研究AI生成图像的巅峰，https://waifulabs.com/，简易的交互式生成模式。

优劣

  • 使用构建好的GAN网络

  • 随机性大，无法精确指定特征，但成图精美

  • 同一个人物有不同标签和动作，方便立绘等衍生商品的制作

  • 无全身立绘，无负面情绪

  • 改进了背景，增加了男性角色

版权

  • 水印下载

DALL·E2

可能是目前来说最好用的生成器，描述语画面内容、参考画风、形式（摄影/绘画/雕塑）。Dall-E 2可以很清晰地生成具象化图片，outpainting使画布无限扩充，修改，甲方的终结者。

优劣

  • 概念、属性和风格结合起来
  • 文本引导进行**图像编辑(**考虑阴影、反射和纹理）
  • 属性绑定对象，unclip模型比GlIDE更差
  • 连贯性较差
  • 很难在复杂场景中生成细节

辅助绘画or替代？

我本人的看法：（已经更改）

现阶段对于娱乐文艺作品的消费是远远大于产能的，能工具的扩大在这种意义上并不可能让画师失业。如果AI能够解决画画中枯燥重复的行为，提供比如灵感，图层筛分，细化，光影，等功能。会是很棒的进步，AI迅速生成庞大数量的次要内容，精细内容再做修改，会帮助很多美术的制作团队，（不是说机器取代了画次要内容的人，而是大家都可以腾出精力去画主要的内容）。

一些非定制化需求的绘画领域比如头像、图标、海报、配图等，会受到冲击。但是同样的，小画师可以在被干掉之前，学习AI，批量生成自己的作品进行售卖，（毕竟会有个人特色），薄利多销呀，况且不是每个人都有调教AI，用大量废图和时间，做没啥特色的AI头像的意愿的。（不然直接搜网图，都不用约稿了）

什么时候翻译集体失业了，画手们也都集体转行去了。喜欢画画并不会因为有一个机器比自己画的更好而绝望，技术的进步是好事，有更多的玩法。

一些评价：

专业艺术家Stigin Windig：作为灵感很不错，但是细节模糊，改画困难不如自己画，没有达到商稿的水平。（另外自己改画的过程将抽象变得具体的同时失去了原有的神秘和冲击感）

西乔（网络画师）：AI能够帮助画师创造完全不同风格的画作（很有趣），用高可控的玩法，设计底图（类似于草稿），或者截断生成过程进行人为干预，后期进行局部合成或者重绘，

网络画师：美术作品的风格总是有迹可循的，因此AI能干完所有的风格；但设计是无穷的，AI干不完所有的设计。因此在未来，美术从业者的核心竞争力就是设计，而非美术风格。

天美Y1工作室的美术总监廖俊豪：AI作画是很棒的工具——它给了非美术专业同学快速通过视觉图像表达自己需求和想法的机会，而且很多时候效果出人意料地好。而对美术来说，它的作用更多体现在更快、更便捷地寻找灵感，或是利用它快速产出灵感方案，以及早期构图、色彩等不同主题的方案。整体而言，用好AI会对原画师提高工作效率有很大帮助，特别是在前期的conception阶段。

  • 底图：害，好多漫画家都直接拍照转背景了

  • 灵感素材：终于要找AI要灵感了

  • 场景概念图的设计：概念，即抽象，在游戏和动漫设计领域的辅助创作

  • 个人风格的底图绘制：学习自己的绘画风格然后节约自己画画初期的时间，详情见下面版权部分。（侵权边缘蹦迪）

  • 枯燥重复：动画制作，live2d ，讲真，画手也希望笔能自动画画啊

艺术

丝毫无法否定AI作画的视觉冲击感。

AI绘画在短短几个月的时间内，几乎翻遍了整个人类的绘画史，很多艺术家也是通过模仿融合借鉴来学习的，AI让这个过程缩短数倍。（不可避免的添加了宗教元素，似乎神秘一点的风格更让人觉得高级？？）很多绘画app就是通过对于某种名家画风的学习创作相似的画作，当然更多的是一些抽象而有冲击力画面构造，用以提供艺术绘画的灵感思路。整体而言，这种艺术创作是抽象的，随机的，需要花大量时间调试的，需要辅助ps或者绘画进行改造的。

随机性意味着出图率和运气有关(很多废图，也要调试)，肯定会有大神调试出最好的画面，得到艺术方面的认可，因为单独从视觉冲击感和碰撞、割裂的感觉来看，AI真的很占优势。（抽象即灵魂？）

AI正在成为艺术品的新变量，影响作品创作风格和价值；以NFT、拍卖画等形式，直接参与艺术品市场交易。

但是不可避免地，AI也学习到了人类认知中对于性别、职业、种族的刻板印象和偏见，这使得开发者们更加谨慎地对待内测，过滤等。但是过滤过度有时候可能妨碍创作。

绘画商业价值

商业约稿的价值一般在几百到数千元，工期为几天到几周不等，绘圈中一般的约稿也会有几十上百等，因为完成一副画作确实需要很多的时间和精力。

劣势：一方面AI生成的高质量绘稿仍然需要付费购买服务，且无法洽谈具体细节（甲方爸爸再也没办法改版了），且很有可能与预期不同，即花钱买一个不确定的东西。另一方面，现有大部分AI作图的细节是不完善的，可以用于海报宣传等概念用途，但更常见的游戏人物设计、场景绘制、卡面游戏绘制至少要更加精细。

优势：很多创作并不需要多精细，比如游戏的素材，勋章，对一些小公司而言，简直绝了。

设计商业价值

适配ppt的完美图片，和设计领域的灵感不谋而合，可以说，科技并没有按我们设想的那样照本宣科毫无灵魂，反而在极端相反的方向狂奔不止

但是谁又阻止了现代设计往古代文化，敦煌啊宗教啥的找灵感呢？

或许以后前期AI，后期PS会称为很多设计师的日常吧

娱乐（和商稿有重合）

无法忽视的一点就是，大众对于这个技术的接纳点到底在哪，会不会变成一种和拍照、ps、滤镜一样的大众的技术。

二次元和文创的需要：对于纸片人的热爱，对于立绘和约稿贼拉贵的痛恨（不过也是在支持原创，暂且不谈），另外很多需要头像、文案配图、自媒体封面、个人创作（oc，小说配图和同人漫画）、装饰画也会带动一部分（甚至相当一部分）的需要，毕竟现在绘圈约图很多不是很好看也很贵还一堆人买。如果结合一些虚拟产业比如原神周边、乙女游戏周边等。

人人绘画的时代？：给了大家一个绘画的可能，但是，对我而言，点点按钮，考我描述能力的绘画过程没有半点成就感。虽然肯定会有一些人喜欢，但是就像拍照一样，当他从一个陌生而有趣的功能变成手机里随意点开的app后，很多人并不会因为会拍照这件事产生热爱，并频繁的去使用这个技能来满足自己的成就感。

色图和本子：可以说这是完全无法避免的一个发展方向，deepfake很火的时候，人们最怕的也是那个一键脱衣的软件，创造一个毫无破绽的deepnude图片，甚至是视频，这是一件很可怕的事情。但是，AI生成将这件事情变得更加的没有成本和可以接受，毕竟人工绘画时代就有不少画黄图，里番，本子一类的画手。目前主流的AI绘图会屏蔽这些关键词。很多色图都没啥人形了，不过想要画总是可以的。

AI作画的版权问题

  1. 2018年一幅由巴黎艺术团体Obvious使用GAN（生成对抗网络）完成的肖像画拍出432500美元。

侵权：

  1. 事实上，真人画手的抄袭也很难界定，很不用说几乎没有什么明确规定的AI了
  2. 如果用别人的作品‘垫’成底图，再把skip开得很高，最终出来的结果，也是很难定义是否构成作品侵权。（听闻还有作品里有原作者的水印hhh）
  3. 已知的一些案例：2022.8.30 micmic（动漫头像生成）的软件在发布的第二天被日本喷到关停，这个软件需要用户上传15到100张插图，进行学习后获得创作者风格的新的插图，即使网站上说你要上传你自己画的插图，但是并没有什么用，依旧被喷死。
  4. 很多画师声明不允许学习自己的画作，有些害怕会在自己的画风作品中看到一些政治因素
